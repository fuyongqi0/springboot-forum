# spring-boot-forum 

### Backend
- Spring Boot 1.5.5
- Spring Security 4
- Mybatis(mybatis-spring-boot-starter 1.3.1)

### Frontend
- Thymeleaf
- Bootstrap
- Chart.js
- bootstrap-select
- daterangepicker

### Build Tool
- Maven 3.3.9

# How To Run
### Run with Maven:
```
mvn spring-boot:run
```
